# Provision Debian `MJML IDE`

Sur une base de l'IDE Atom, provisionné avec  https://github.com/Jean-Baptiste-Lasselle/provision-debian-atom-ide, cette recette 
spécialise Atom, avec 3 plugins, pour en faire un éditeur MJML, assité, et avec aperçu reactif : 

* https://github.com/mjmlio/language-mjml/releases _(coloration syntaxique MJML, et peut-être certains traitements pour la génération du HTML)_
* https://github.com/mjmlio/atom-mjml-preview/releases _(aperçu graphqiue des MJML)_
* https://github.com/mjmlio/atom-linter-mjml  _(intellisense pour MJML avec un `linter`)_
